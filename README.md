# jsx-suit
Quick-start suit for JSX development

Includes:
  - Grunt task-runner
  - Webpack (with dev-server)
  - React
  - ES6 and JSX compiler (babel)
  - LESS compiler
  - Nodemon server
  
Quick start:
  - npm start
  - grunt - will start locahost:1234 server and webpack with watch option
  - grunt hot - will start localhost:4321 hot-server
